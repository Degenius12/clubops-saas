## 🔗 How to Access Files

Each document entry has a **File Path** field showing where the actual file lives:

**Examples**:
- `marketing/QUICK-START-GUIDE.md` → Open in your code editor
- `~/.claude/plans/agile-enchanting-turtle.md` → Claude plan file
- `docs/CSV_IMPORT_FEATURE_SPEC.md` → Technical spec

**To Open**:
1. Copy the file path from Notion
2. Open your IDE/editor
3. Use Cmd/Ctrl + P → Paste path → Open

## 🆕 Adding New Documents

When you create new marketing materials, guides, or docs:

1. **Click "+ New" in the Documents database**
2. **Fill in**:
   - Document Name: Clear, descriptive title
   - Category: Marketing/Technical/etc.
   - Type: Guide/Template/Specification/etc.
   - Priority: How important is it?
   - Status: Active/Draft/Archived
   - File Path: Where it lives in your repo
   - Last Updated: Today's date

3. **Save** - It's now searchable and organized!

## 💡 Pro Tips

### 1. Star Your Most-Used Docs
- Click "..." on any document
- Add to Favorites
- Appears in your Notion sidebar

### 2. Use Table View for Scanning
- Switch to Table view (top right)
- See all docs at once
- Sort by Priority or Category

### 3. Link to Docs from Tasks
- In your Tasks database
- Add document links in Notes field
- Example: "Follow email template: [link]"

### 4. Weekly Review
- Every Friday, review "Critical" docs
- Update status (Active → Archive old ones)
- Add new docs you created this week

### 5. Mobile Access
- Install Notion mobile app
- Access all docs on the go
- Read marketing plan while traveling

### When You Create New Marketing Materials:

2. Include file path
3. Tag with appropriate category

1. Find it in Notion
2. Update "Last Updated" date
3. Change status if needed (Draft → Active)

### When You Archive Old Docs:
1. Don't delete (keep for reference)
2. Change Status → "Archived"
3. Change Priority → "Low"
4. Filter them out of daily views
