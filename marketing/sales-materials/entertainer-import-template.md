## Instructions for Club Managers

### Step 1: Make a Copy
1. Click the template link above
2. File → Make a copy
3. Rename: "[Your Club Name] Entertainers"

### Step 2: Fill Out Data

**Required Fields** (MUST fill these):
- **Stage Name**: The name they use on stage (e.g., "Sasha", "Diamond")
- **First Name**: Legal first name
- **Last Name**: Legal last name
- **Phone**: Cell phone number

**Optional Fields** (fill if you have it):
- **Email**: Personal email address
- **Date of Birth**: For age verification (format: MM/DD/YYYY)
- **License Number**: Entertainer license number
- **License State**: State that issued license (2-letter code: CA, NV, TX, etc.)
- **Emergency Contact**: Name of emergency contact person
- **Emergency Phone**: Emergency contact phone number
- **Notes**: Any special notes (preferred shifts, VIP only, etc.)

### Step 3: Format Examples

**Phone Numbers**:
- ✅ Good: (555) 123-4567 or 555-123-4567 or 5551234567
- ❌ Bad: 555.123.4567

**Dates**:
- ✅ Good: 01/15/1995 or 1/15/1995
- ❌ Bad: Jan 15 1995, 1995-01-15

**License State**:
- ✅ Good: CA, NV, TX, FL
- ❌ Bad: California, Nevada

### Step 4: Review Checklist

Before uploading, verify:
- [ ] All stage names are unique (no duplicates)
- [ ] All phone numbers have 10 digits
- [ ] All required fields are filled
- [ ] No empty rows in the middle
- [ ] Column headers haven't changed

### Step 5: Download as CSV

1. File → Download → Comma Separated Values (.csv)
2. Save to your computer
3. Remember where you saved it!

### Step 6: Upload to ClubFlow

1. Log into ClubFlow
2. Go to: Dashboard → Settings → Import Entertainers
3. Click "Choose File" and select your CSV
4. Click "Upload and Preview"
5. Review the preview (fix any errors)
6. Click "Confirm Import"

## Common Errors & Fixes

### Error: "Duplicate stage name"
**Problem**: Two entertainers have same stage name
**Fix**: Add last initial or number (e.g., "Sasha M" or "Angel 2")

### Error: "Invalid phone number"
**Problem**: Phone number doesn't have 10 digits
**Fix**: Add area code or remove extra characters

### Error: "Invalid date format"
**Problem**: Date isn't recognized
**Fix**: Use MM/DD/YYYY format (e.g., 01/15/1995)

### Error: "Missing required field"
**Problem**: Required column is empty
**Fix**: Fill in Stage Name, First Name, Last Name, and Phone for all rows

## Tips for Faster Entry

1. **Copy from existing spreadsheet**: If you already have data, copy/paste columns
2. **Use autocomplete**: For License State, type first letter and Excel suggests
3. **Fill down**: If many entertainers have same state, fill first cell and drag down
4. **Work during slow shift**: Have a manager do this during 2-5pm slow time
5. **Divide and conquer**: One person does A-M, another does N-Z
